About This Project

The Green EV Charging Station is a sustainable and intelligent electric vehicle (EV) charging system designed using Arduino Uno. It primarily utilizes solar power to charge electric vehicles, promoting renewable energy usage and reducing dependency on conventional power sources. In cases where solar energy is insufficient or unavailable (such as during cloudy weather or nighttime), the system automatically switches to a battery backup to ensure uninterrupted charging.

To enhance security and personalization, the system integrates NFC (Near Field Communication) authentication, allowing only authorized users to initiate the charging process. Once authenticated, the system begins charging and actively monitors and records key metrics such as charging time and current consumed. This data can be accessed through an IoT-enabled interface, allowing users to view real-time information on a smartphone or connected display.

Built with C++ on the Arduino platform, the system employs a modular and object-oriented approach for handling hardware components like the solar panel, battery management system, NFC reader, and sensors. This project not only supports eco-friendly transportation but also demonstrates the integration of automation, security, and real-time monitoring in modern renewable energy systems.

Technology Used

Technologies used in this project - C++, Circuits, Solar, Sensors, Arduino Uno & libraries.
